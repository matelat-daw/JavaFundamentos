Comandos Maven:

mvn eclipse:eclipse // Genere Archivos .project y .classpath

mvn clean // Limpia el Proyecto, Elimina la Carpeta target

mvn package // Crea el la Carpeta target y el Paquete war. Para funcionar tiene que estar la etiqueta: <packaging>war</packaging> en el pom.xml

Se peuden Usar Juntos: mvn clean package

MariaDB SQL:

-- Crea el usuario sin contraseña y permite conexiones desde cualquier IP (%)
CREATE USER 'rooot'@'%' IDENTIFIED VIA mysql_native_password USING '';

-- Otorga todos los privilegios sobre cualquier base de datos y tabla (*.*)
GRANT ALL PRIVILEGES ON *.* TO 'rooot'@'%' WITH GRANT OPTION;

-- Recarga los privilegios en la memoria del servidor
FLUSH PRIVILEGES;
